Pastille - More convenient npm install && bower install

I got tired of running npm install && bower install over and over again so I wrote a bash script to do it for me automatically. Read the explanation below, or just copy it to your path! :)

A lot of times, I found myself quickly switching between git branches - and even differnt node and npm versions - with totally different dependencies. Switching back and forth and doing npm installs multiple times a day got cumbersome, so I wrote that little script.

What the script does

Basically it just makes backups of node_modules/ and vendor/ saving them under ~/.pastille and the other way around.

  1. Calculates a checksum of package.json, your node and npm version. (It uses either md5 or md5sum = should work on both OSX and Linux)
  2. Checks from ~/.pastille/ if a backup file for this combination exists.
    3a. If exists, uncompress the backup file into the current directory.
    3b. If not, runs npm install and backups the node_modules directory under .pastille/ with the hash.
  3. Do the same with bower’s files.

Installation & Usage

Just copy it to your path, eg. /usr/local/bin as ‘pastille’, and use it instead of npm install :)
